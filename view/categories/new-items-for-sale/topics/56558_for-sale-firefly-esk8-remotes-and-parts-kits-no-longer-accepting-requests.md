## \#318 Posted by: SeeTheBridges Posted at: 2018-08-13T22:27:07.156Z Reads: 187

```
Apologies everyone, the reply notifications for this thread didn't make its way to my inbox. So setup wise, There's a couple things that need to be done. 

On the remote side, when powering on, press the trigger and push forward on the throttle to get to the settings menu. Here you can change the control style (ONLY WHEN CONNECTED TO THE RECEIVER), battery setup (# of cells and type), gear ratios, and calibrate the hall sensor values. I usually pre calibrated remotes before I shipped them, but I may have missed a few so it doesn't hurt to check. The options after the hall sensor values have to do with the PPM widths, don't mess with those. Then there's the generate new address and reset address functions. I'll be honest these don't always work that great so just avoid them. you can exit the settings on the last menu option. 

On to the receiver side, every remote and receiver is flashed with matching addresses, so as long as you didn't touch the generate address function on the remote, they should auto pair when powered on. IF you did touch the auto generate address function and it's no longer pairing, you'll need to refresh your receiver with the matching code, or just wipe both of them and flash again to get them to match.

On to the VESC side of the setup. With whatever VESC tool your firmware uses: BLDC tool, Ackmaniac's ESC tool, etc, you need to set your control type to "PPM and UART". Along with this, you need to find the baud rate setting for UART and set it to 115200. Last thing you'll need to do is find the PPM calibration page. With the remote already calibrated, just follow the procedure for setting your min, max, and center PPM values. After that, you should be ready to go. 

Really wish I had some more time to put the video version of this out, but moving in 3 days has me scrambling!
```
